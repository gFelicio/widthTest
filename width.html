NOTE: The first image will break the layout when it goes below 150px wide.

NOTE2: Usually, the layout will break when the window width can't fit the biggest word in this page contained inside itself. BUT, with the property overflow-wrap: break word; in our CSS file, words can't break the layout.

NOTE: The AUTO definition BREAKS the entire page layout because it takes the original width of the image and keeps it this way, so the minimum width of the page must always be the minimum width of the smallest image in the page.

TL;DR: Pixels and auto-sizes can break my layout, but words can never break them.
